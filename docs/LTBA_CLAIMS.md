# LTBA Claims

## Supported

- LTBA preserves tested equivalence with explicit Piecewise semantics.
- LTBA avoids eager global branch materialization.
- LTBA preserves local branch provenance.
- LTBA wins v4 fair amortized runtime for nontrivial guarded benchmark families.

## Not claimed

- LTBA does not universally beat BDDs.
- LTBA does not replace Mathematica, SymPy, Sage, or Piecewise.
- LTBA does not prove a new complete algebra.
- LTBA is not yet independently reviewed.

## Main open question

Can LTBA be formalized as a CAS intermediate representation and integrated into existing symbolic systems as a pre-Piecewise guarded-local normal form?
